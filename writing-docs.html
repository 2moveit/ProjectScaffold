Writing docs

In order to edit the docs run:

1: 

$ `build KeepRunning`

  • open docs/output/index.html
  • edit the files in docs/content - the build script will update the output automatically
  • refresh browser
  • press any key in command line window when you are ready.

Releasing docs

In order to release the docs:

  • run build ReleaseDocs
